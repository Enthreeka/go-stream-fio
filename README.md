

Развертывание dev среды:

`make docker-up`

Закрытие dev среды:

`make docker-down`

Запуск главного сервера:

`make server` 

Запуск producer сервера:

`make producer`

Создание миграций:

`make migrate-up`

Дроп таблиц:

`make migrate-down`
